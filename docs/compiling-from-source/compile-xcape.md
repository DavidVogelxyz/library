How to compile `xcape` on Rocky Linux
=====================================

Date added: 2025 December 25, Thursday

Notes
-----

Dependencies:

```
libXi-devel
libXtst-devel
```

Clone the repo:

```bash
git clone https://github.com/alols/xcape
```

Change directory into the repo. Then, run the following commands:

```bash
make
sudo make install
```
